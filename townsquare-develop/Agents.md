# Agents.md

## 适用范围

本文件适用于 `townsquare-develop/` 目录及其子目录。这里是 Vue 2 版 Town Square Web 项目，开发时同时遵守仓库根目录的 `AGENTS.md`。

## PowerShell 使用要求

- 在本项目中使用 PowerShell 执行命令、写说明或汇报结果时，优先使用中文表达。
- 命令本身保持原样，例如 `npm run lint`、`npm run build`。
- 如果 PowerShell 输出中文乱码，优先确认文件编码或使用 `-Encoding UTF8` 读取，不要直接判断文件内容已损坏。

## 多语言要求

- 新增或修改面向用户的功能时，必须同步考虑多语言。
- 不要在 Vue 模板、组件逻辑、弹窗、按钮、提示、错误消息中直接硬编码用户可见文案。
- 优先使用项目现有 i18n 机制，消息集中维护在 `src/i18n/index.js`。
- 新增文案时至少补齐 `zh-CN` 和 `en-US` 两套 key，并在组件里通过 `$t(...)` 或项目现有翻译调用方式引用。
- 修改已有功能文案时，检查对应中英文翻译是否都需要同步更新。
- 如果某些内容来自服务端或外部数据，无法在前端 i18n 中维护，应在最终回复里说明原因。

## 验证建议

- 修改源码后优先运行 `npm run lint` 或 `npm run lint-ci`。
- 修改构建配置或依赖相关内容后运行 `npm run build`。
- 如果只修改本说明文件，不需要运行构建或 lint。
