# 用户故事1 - 搜索和筛选手动测试流程

**用户故事**: US1 - 小程序端剧本数据展示
**测试目标**: 验证搜索和筛选功能正常工作
**优先级**: P1 (MVP)
**预计测试时间**: 8分钟

## 测试前准备

### 测试数据准备
确保数据库中有不同标题和作者的剧本：
- "狼人杀剧本" - 作者："官方"
- "推理游戏" - 作者："社区"
- "欢乐聚会" - 作者："官方"

## 测试步骤

### 步骤1: 标题搜索测试
1. 点击搜索框
2. 输入"狼人杀"
3. 确认搜索结果

**预期结果**:
- ✅ 只显示包含"狼人杀"的剧本
- ✅ 搜索结果实时更新
- ✅ 可以清除搜索条件

### 步骤2: 作者搜索测试
1. 清除搜索条件
2. 输入"官方"
3. 确认搜索结果

**预期结果**:
- ✅ 显示所有作者为"官方"的剧本
- ✅ 多个结果正确显示

### 步骤3: 无结果搜索测试
1. 输入不存在的关键词"不存在的剧本"
2. 确认显示"无结果"状态

**预期结果**:
- ✅ 显示友好提示
- ✅ 可以重新搜索

## 验收标准

- [ ] 搜索结果准确无误
- [ ] 搜索响应及时（<1秒）
- [ ] 无结果时有适当提示
- [ ] 可以清除搜索条件
