# 血染钟楼微信小程序

一个用于浏览和分享血染钟楼游戏剧本的微信小程序。

## 功能特性

### 🎭 剧本浏览
- 展示所有可用剧本
- 支持剧本搜索（按名称或作者）
- 显示剧本基本信息（名称、作者、版本号）
- 复制JSON地址功能
- 0-3张图片轮播展示

### 📄 剧本详情
- 完整的剧本信息展示
- 图片轮播
- 角色列表和能力说明
- 点赞功能
- 使用统计

### 🏆 排行榜
- 使用次数排行
- 点赞数量排行
- 清晰的排名显示（金银铜牌样式）

### 👤 我的页面
- 用户信息展示
- 投稿须知弹窗
- 其他功能入口（收藏、统计、设置）

## 项目结构

```
xueran/                    # 小程序主项目
├── pages/                 # 页面文件
│   ├── script-list/       # 剧本浏览页面
│   ├── script-detail/     # 剧本详情页面
│   ├── rankings/          # 排行榜页面
│   └── profile/           # 我的页面
├── static/                # 静态资源
├── pages.json             # 页面配置和标签栏
├── manifest.json          # 小程序配置
└── App.vue                # 应用入口

xueran-admin/              # 管理后台（可选）
```

## 开发环境

- HBuilderX 3.0+
- uni-app框架
- Vue 2.x

## 运行项目

1. 使用HBuilderX打开项目
2. 选择`xueran`目录作为项目根目录
3. 运行 -> 运行到浏览器(H5) 或 运行到微信开发者工具

## 页面说明

### 剧本浏览页面 (script-list)
- 搜索功能：支持按剧本名称或作者搜索
- 列表展示：卡片式布局，包含图片轮播
- 复制功能：一键复制剧本JSON地址

### 剧本详情页面 (script-detail)
- 详细信息展示：标题、作者、版本、描述等
- 图片轮播：最多支持3张图片
- 角色信息：完整的角色列表和能力说明
- 互动功能：点赞、复制JSON地址

### 排行榜页面 (rankings)
- 双标签页：使用排行和点赞排行
- 排名显示：1-3名有特殊样式
- 点击跳转：点击可跳转到对应剧本详情

### 我的页面 (profile)
- 用户信息：显示当前用户状态
- 投稿须知：弹窗形式展示详细规则
- 功能入口：收藏、统计、设置等

## 数据格式

剧本数据采用JSON格式，包含以下字段：

```json
{
  "id": 1,
  "title": "剧本标题",
  "author": "作者名称",
  "version": "1.0.0",
  "updateTime": "2024-01-12",
  "description": "剧本描述",
  "playerCount": "8-12人",
  "duration": "30-45分钟",
  "difficulty": "简单",
  "usageCount": 1250,
  "likes": 156,
  "jsonUrl": "https://example.com/script.json",
  "images": ["url1", "url2", "url3"],
  "roles": [
    {
      "id": 1,
      "name": "角色名称",
      "team": "狼人/村民",
      "description": "角色能力描述"
    }
  ]
}
```

## 注意事项

- 当前版本为演示版本，使用模拟数据
- 图片路径需要根据实际部署情况调整
- 标签栏图标暂时移除，可根据需要添加
- 投稿须知内容暂定，可根据实际需求修改
